*   **출력:** 백엔드 API 엔드포인트 (예: `/api/status`)를 통해 실시간으로 PTI와 Safe Mode 상태를 프론트엔드로 전송.

#### 2. 핵심 코드 구조 구체화 (Core Code Specification)
