## 🖥️ 최종 UI/UX 디자인 및 프로토타입 구현 사양

### 1. 디자인 철학 및 컬러 가이드라인 재확인 (Design Philosophy & Color Guide)

*   **핵심 철학:** 안정성(PTI)이 수익률(ROI)보다 우선한다.
*   **메인 컬러:**
    *   **Primary Background:** Deep Navy Blue (`#0A1931`) - 깊은 신뢰감과 전문성 표현.
    *   **Stability Indicator (PTI):** System Green (`#4CAF50`) - 시스템 안정성과 긍정적 상태 강조.
    *   **Warning/Risk Indicator:** Red (`#F44336`) - 안전 모드 활성화, 위험 경고 시 사용.
    *   **Accent:** Light Gray/White (`#E0E0E0`) - 데이터 가독성 확보.

### 3. 프로토타입 구현 코드 (Prototype Implementation Code)

이 사양을 바탕으로, 핵심 지표(PTI)의 우선순위를 반영한 React/Tailwind CSS 기반의 대시보드 컴포넌트 코드를 생성합니다.

import React, { useState } from 'react';

// --- Color Palette Definition (Based on Designer Guidelines) ---
const COLORS = {
  NAVY_BLUE: '#0A1931', // Primary Background
  SYSTEM_GREEN: '#4CAF50', // Stability Indicator (PTI Green)
  RED_ALERT: '#F44336',   // Risk Alert (Safe Mode)
  TEXT_LIGHT: '#E0E0E0', // Text/Accent
};

// --- Mock Data for Demonstration ---
const mockData = {
  pti: 98.5, // System Stability Index (PTI)
  roi: 1.25, // Return on Investment
  safeMode: false,
  status: "System Stable",
};

// --- Core Dashboard Component ---
const DashboardPrototype = () => {
  const [data, setData] = useState(mockData);

  // Dynamic color based on PTI status
  const ptiColor = data.pti >= 90 ? COLORS.SYSTEM_GREEN : COLORS.RED_ALERT;
  const headerBg = data.safeMode ? COLORS.RED_ALERT + '20' : COLORS.NAVY_BLUE; // Subtle background change for alert

  return (
    <div style={{ backgroundColor: COLORS.NAVY_BLUE, minHeight: '100vh', color: COLORS.TEXT_LIGHT, fontFamily: 'Arial, sans-serif' }}>
      
      {/* A. 최상단 시스템 상태 표시줄 (PTI Emphasis) */}
      <header style={{ 
        backgroundColor: headerBg, 
        padding: '20px', 
        textAlign: 'center', 
        borderBottom: `4px solid ${ptiColor}` 
      }}>
        <h1>갓더주식 대시보드</h1>
        <div style={{ fontSize: '3em', fontWeight: 'bold', color: ptiColor, margin: '10px 0' }}>
          PTI: {data.pti.toFixed(2)}
        </div>
        <p style={{ color: data.safeMode ? COLORS.RED_ALERT : COLORS.SYSTEM_GREEN, fontWeight: 'bold' }}>
          상태: {data.status}
        </p>
      </header>

      {/* B. 메인 대시보드 영역 */}
      <main style={{ padding: '30px' }}>
        <h2>투자 성과 요약</h2>
        
        {/* KPI Card 1: PTI (Most Prominent) */}
        <div style={{ 
          backgroundColor: '#1A2B47', // Slightly lighter than background
          padding: '25px', 
          borderRadius: '10px', 
          marginBottom: '20px',
          borderLeft: `5px solid ${ptiColor}` // Green border emphasis
        }}>
          <h3>시스템 안정성 지수 (PTI)</h3>
          <p style={{ fontSize: '3em', margin: '5px 0' }}>{data.pti.toFixed(2)}</p>
          <p>안정성이 곧 수익입니다.</p>
        </div>

        {/* KPI Card 2: ROI */}
        <div style={{ 
          backgroundColor: '#1A2B47', 
          padding: '25px', 
          borderRadius: '10px', 
          borderLeft: `5px solid ${data.pti >= 90 ? COLORS.SYSTEM_GREEN : COLORS.RED_ALERT}`
        }}>
          <h3>투자 수익률 (ROI)</h3>
          <p style={{ fontSize: '3em', margin: '5px 0' }}>{data.roi.toFixed(2)}</p>
          <p>현재 시장 대비 성과 지표.</p>
        </div>

        {/* C. 리스크 섹션 */}
        <div style={{ marginTop: '30px', padding: '20px', borderRadius: '10px' }}>
          <h3>리스크 제어 상태</h3>
          {!data.safeMode ? (
            <p style={{ color: COLORS.SYSTEM_GREEN, fontSize: '1.2em', fontWeight: 'bold' }}>
              ✅ 안전 모드 해제됨 (System Stable)
            </p>
          ) : (
            <p style={{ color: COLORS.RED_ALERT, fontSize: '1.2em', fontWeight: 'bold' }}>
              🚨 **경고: 안전 모드 활성화됨** - 자동매매 일시 정지
            </p>
          )}
        </div>

      </main>
    </div>
  );
};

export default DashboardPrototype;

## 📊 PTI와 ROI의 수학적 연결 고리 분석 및 기술 보고서

### 1. 핵심 철학 요약: 안정성이 수익률보다 우선한다 (Stability Precedes Profit)

우리의 플랫폼은 단순한 투자 수익(ROI) 극대화를 목표로 하지 않습니다. 우리의 핵심 가치는 **'리스크 제어형 스마트 투자 플랫폼'**으로, 이는 시스템의 **절대적 안정성($PTI$) 확보가 모든 수익 창출 활동의 선행 조건**임을 의미합니다.

*   **핵심 전제:** $ROI$는 $\text{Stability} \times \text{Execution Quality}$에 의해 결정된다.
*   **우선순위:** 시스템 안정성 지수($PTI$) $\gg$ 수익률($ROI$).

### 2. 수학적 연결 고리 (The Mathematical Link)

시스템 안정성 지수($PTI$)는 자동매매의 안전성과 데이터 무결성을 측정하는 핵심 변수이며, 이는 최종 수익($ROI$)에 직접적인 제약 조건으로 작용합니다.

**연결 로직:**
$$
\text{Effective ROI} = \text{Gross ROI} \times (1 - \text{Risk Penalty})
$$

여기서 $\text{Risk Penalty}$는 시스템 안정성 지수($PTI$)가 설정된 임계치를 초과할 경우, 자동매매 실행에 적용되는 페널티입니다.

**$PTI$의 역할:**
$PTI$는 데이터 무결성 실패 또는 리스크 헷지 모듈(`risk_hedge_integration.py`)의 안전장치(`system_safe_mode`)가 활성화되었는지 여부를 반영합니다.

*   만약 $PTI \ge \text{Threshold}_{\text{Safe}}$ 이면, $\text{Risk Penalty} = 1$ (자동매매 정지 및 손실 제한 강제 실행)
*   만약 $PTI < \text{Threshold}_{\text{Safe}}$ 이면, $\text{Risk Penalty} = 0$ (정상적인 수익 계산 진행)

**기술적 검증 결과 (자율 사이클 기반):**
이 논리는 `risk_hedge_integration.py` 내의 중앙 제어 함수(`execute_hedge_logic`)에 완벽하게 반영되었으며, 데이터 무결성 실패 시 $system\_safe\_mode$가 즉각적으로 활성화되어 모든 자동매매 및 리스크 계산 흐름이 중단되고 손실 제한 메커니즘이 강제 실행됨을 End-to-End 테스트를 통해 확인했습니다. [근거: sessions/2026-05-14T09-49/developer.md, sessions/2026-05-14T10-04/developer.md]

### 3. 기술적 근거 및 시스템 안정성 확보 방안

**A. 데이터 무결성 검증 모듈의 역할:**
키움 RestApi로부터 수신되는 모든 주식 데이터에 대해, $F1$ 모듈에서 정의된 기준을 적용하여 실시간으로 데이터 무결성을 검사합니다. 이 단계에서 오류가 발생하면 시스템은 즉시 $\text{system\_safe\_mode}$로 전환됩니다. [근거: sessions/2026-05-14T06-09/developer.md]

**B. 리스크 헷지 모듈의 통합:**

**C. 시스템 안정성 지수($PTI$)의 시각화 및 통제:**

## 1. 브랜드 비주얼 시스템 가이드라인 (Visual System Guideline)

이 가이드라인은 모든 콘텐츠(영상, 썸네일, 대시보드)에서 일관된 신뢰감과 리스크 제어형 플랫폼의 정체성을 전달하는 데 사용됩니다.

### 🎨 핵심 컬러 팔레트 (Core Color Palette)

시스템 안정성과 신뢰감을 극대화하기 위해 아래 두 가지 색상을 중심으로 모든 디자인 요소를 구성합니다.

| 역할 | 색상 이름 | HEX 코드 | 용도 및 의미 |
| :--- | :--- | :--- | :--- |
| **Primary (안정성)** | Deep Navy Blue | `#0A1931` | 플랫폼의 깊이, 전문성, 시스템 안정성을 상징. 배경 및 주요 텍스트에 사용. |
| **Accent (성장/안전)** | System Green | `#16A085` | 성공, 성장, 안전 모드 활성화(Safe Mode), 긍정적인 결과 표시. 강조 요소에 사용. |
| **Secondary (보조)** | Light Gray | `#F0F4F8` | 배경 및 보조 정보 영역. 데이터의 명확성을 높임. |
| **Alert (경고)** | Red Alert | `#FF3B30` | 위험, 경고, 시스템 안전 모드 비상 상황을 표시. 최소한으로 사용하며 강력한 대비를 제공. |

### ✒️ 타이포그래피 스타일 (Typography Style)

데이터 기반의 명확성과 전문성을 강조합니다.

*   **주요 제목/핵심 메시지:** **Pretendard Bold** 또는 **Noto Sans Bold** (강력하고 현대적인 느낌).
*   **본문/데이터 표시:** **Noto Sans Regular** (가독성이 높고 데이터 전달에 최적화됨).

### 🖼️ 레이아웃 및 스타일 규칙 (Layout & Style Rules)

1.  **배경 우선 원칙:** 모든 주요 콘텐츠(대시보드, 영상 배경)는 **Deep Navy Blue**를 주조색으로 사용하여 깊이감과 신뢰감을 확보합니다.
2.  **액션 강조:** 사용자가 취해야 할 핵심 행동(CTA), 성공 지표(ROI 상승, 안정성 확보)는 **System Green**을 사용하여 즉각적인 주목도를 높입니다.
3.  **경고 시스템 (Safe Mode):** `system_safe_mode`가 활성화되는 상황은 **Red Alert** 색상을 사용하여 즉시 인식하게 하며, 이는 시스템의 통제권을 사용자에게 되돌려준다는 메시지를 시각적으로 전달합니다.
4.  **데이터 시각화:** 모든 차트와 그래프는 명확하고 간결해야 합니다. 복잡한 데이터는 단순화하여 '시스템 안정성 지수(PTI)'를 중심으로 해석되도록 디자인합니다.

## 2. 썸네일 컨셉 3안 (Thumbnail Concepts)

제시된 비주얼 시스템을 적용하여, 유튜브 및 소셜 미디어에서 **'시스템 안정성'**과 **'리스크 제어'**라는 핵심 메시지를 극대화하는 썸네일 초안 3가지를 제안합니다.

### 💡 컨셉 A: [PTI 우선 강조형] - 신뢰와 전문성 어필
*   **주요 메시지:** 수익률보다 중요한 것, 시스템 안정성(PTI)의 중요성.
*   **스타일:** 심플하고 전문적인 데이터 중심.
*   **레이아웃:**
    *   배경: Deep Navy Blue (어두운 톤).
    *   중앙: 거대한 `PTI > ROI` 비교 그래프 (System Green으로 PTI를 강조).
    *   텍스트: "수익률보다 중요한 것" (크고 흰색), "시스템 안정성(PTI)이 왕이다" (강조).
    *   하단: 갓더주식 로고 및 작은 System Green 안전 아이콘.
*   **키워드:** 신뢰, 철학, 전문성.

### 💡 컨셉 B: [안전 모드 강조형] - 즉각적인 통제감 어필
*   **주요 메시지:** 시스템 안전 모드(`system_safe_mode`)를 통한 리스크 제어.
*   **스타일:** 긴급성과 해결책 제시.
*   **레이아웃:**
    *   배경: Deep Navy Blue.
    *   중앙: 빨간색 경고등(Red Alert) 아이콘과 함께 **"SYSTEM SAFE MODE"** 문구 강조.
    *   강조: System Green으로 '안전 확보'의 의미를 표현하는 자물쇠 또는 방패 아이콘 배치.
    *   텍스트: "자동매매 멈춤, 내 자산 보호" (명확한 행동 유도).
*   **키워드:** 통제감, 안전, 즉각적 해결.

### 💡 컨셉 C: [비교 및 차별화형] - 티어(Tier) 비교 어필
*   **주요 메시지:** Basic vs Pro 티어 간의 시스템 안정성 차이점 시각화.
*   **스타일:** 명확한 구분과 가치 제시.
*   **레이아웃:**
    *   배경: Deep Navy Blue.
    *   구조: 화면을 좌우 또는 상하로 분할하여 Basic/Pro 티어를 명확히 구분.
    *   각 티어 옆에 시스템 안정성 지수(PTI)를 막대 그래프로 표시하고, Pro 티어가 더 높은 System Green 수치를 가짐을 강조.
    *   텍스트: "Basic vs Pro: 리스크 관리의 차이" (중앙), 각 티어별 핵심 이점 명시.
*   **키워드:** 비교, 단계적 성장, 프리미엄 가치.
