### 🖼️ 최종 Error State 디자인 확정 (Error State Specification)

모든 에러는 세 가지 주요 카테고리로 분류되며, 각 상황에 따라 색상과 메시지 톤을 다르게 적용하여 사용자가 혼란 없이 대응할 수 있도록 설계합니다.

#### 1. 정보/입력 오류 (Input/Validation Errors: HTTP 4xx)
**대상:** 사용자 입력 또는 요청 파라미터의 문제 (Test 2, Test 4)
**목표:** 즉각적인 수정 및 재시도를 유도.
**디자인 원칙:** 경고(Warning) 색상 사용.

| 오류 유형 | HTTP 코드 | 표시 컨테이너 색상 | 타이포/아이콘 | 에러 메시지 (사용자에게 보일 내용) | 상세 설명 (개발자용) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **필터 조건 오류** | 400 Bad Request | `#FFC107` (Amber/Warning) | ⚠️ 아이콘 + Bold Text | "필터 조건 형식이 잘못되었습니다. '기간' 값이 누락되었습니다." | `Request Parameter Error` |
| **권한 부족 오류** | 403 Forbidden | `#F44336` (Red/Error) | 🔒 아이콘 + Bold Text | "접근 권한이 부족합니다. API 키 또는 세션 정보가 유효하지 않습니다." | `Authentication Failed` |

#### 2. 서버 상태 오류 (Server Status Errors: HTTP 5xx)
**대상:** 시스템 내부 문제 또는 서비스 일시 중단 (Test 3, Test 6)
**목표:** 서비스 자체의 일시적 문제임을 알리고 사용자에게 인내를 요청.
**디자인 원칙:** 심각성(Severity)을 나타내는 빨간색 계열 사용.

| 오류 유형 | HTTP 코드 | 표시 컨테이너 색상 | 타이포/아이콘 | 에러 메시지 (사용자에게 보일 내용) | 상세 설명 (개발자용) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **서비스 일시 중단** | 503 Service Unavailable | `#FF9800` (Orange/Caution) | 🛑 아이콘 + Bold Text | "서버 부하로 인해 현재 예측 서비스가 일시 중단되었습니다. 잠시 후 재시도해 주십시오." | `Server Error` |
| **내부 서버 오류** | 500 Internal Server Error | `#D32F2F` (Deep Red/Critical) | 💥 아이콘 + Bold Text | "스트리밍 데이터 처리 중 내부 서버 오류가 발생했습니다. 시스템 점검 중입니다." | `Internal Server Error` |

#### 3. 성공 및 일반 오류 (Success & General State)
**대상:** 정상적인 데이터 수신 또는 기타 예상치 못한 상황 (Test 1, Test 5)
**목표:** 긍정적 피드백 제공.
**디자인 원칙:** 녹색 계열 사용.

| 상태 | HTTP 코드 | 표시 컨테이너 색상 | 타이포/아이콘 | 에러 메시지 (사용자에게 보일 내용) | 상세 설명 (개발자용) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **성공** | 200 OK | `#4CAF50` (Green/Success) | ✅ 아이콘 + Bold Text | "데이터 수신 및 포맷 검증 성공." | `Success` |

### 💻 UI/UX 흐름 반영 요약

1.  **상황별 색상 매핑:** 에러의 성격(클라이언트 오류 vs 서버 오류)에 따라 명확하게 구분되는 색상을 사용하여 사용자가 직관적으로 문제의 종류를 파악하도록 합니다.
2.  **액션 유도:** 4xx 오류(사용자 입력 실수)는 '수정'을, 5xx 오류(서버 문제)는 '재시도' 또는 '잠시 후 확인'을 유도하는 명확한 Call to Action (CTA) 버튼을 함께 배치합니다.
3.  **구체적인 메시지:** 시스템이 반환한 기술적 에러 코드 대신, 사용자가 이해할 수 있는 구체적인 해결책을 제시하도록 메시지를 최적화했습니다.
