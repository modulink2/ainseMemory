## 🎯 대시보드 디자인 및 UX 우선순위 최종 확정

**핵심 원칙:** **신뢰(Trust) 시각화** — 사용자가 시스템의 안정성을 즉각적으로 인지하고, 데이터에 기반한 의사결정을 내릴 수 있도록 명확성과 직관성을 최우선으로 합니다.

### 1. 대시보드 전체 디자인 사양 (Global Specs)

| 항목 | 세부 내용 | 적용 근거 |
| :--- | :--- | :--- |
| **브랜드 컬러 팔레트** | **Primary (신뢰):** `#007bff` (Deep Blue), **Secondary (안정):** `#28a745` (Success Green), **Warning (경고/주의):** `#ffc107` (Amber), **Danger (위험):** `#dc3545` (Red) | 갓더주식의 전문성과 금융 시장의 신뢰를 반영하며, 색상 대비를 통해 KPI 상태를 명확히 구분합니다. |
| **타이포그래피** | **Font Family:** Pretendard (또는 Inter), **Header:** Bold, Large Size (H1, H2) / **Data Labels:** Regular Weight, Clear Contrast | 금융 데이터의 정확성과 가독성을 극대화하며, 복잡한 수치 정보 속에서 핵심 지표를 부각합니다. |
| **레이아웃 구조** | **3단 분할 구조:** 좌측 사이드바 (Navigation/KPI 요약) / 중앙 메인 영역 (핵심 시각화 대시보드) / 우측 (상세 로그 및 알림) | `DashboardKit`의 논리를 따르되, 핵심 KPI를 중앙에 배치하여 사용자의 시선이 가장 먼저 안정성 지표에 꽂히도록 설계합니다. |

### 2. 핵심 컴포넌트 UI/UX 우선순위 재점검 (Priority Re-evaluation)

**최우선 목표:** **시스템 신뢰도 즉각 전달**을 위해, 사용자가 시스템 상태를 파악하는 데 필요한 최소한의 정보(Minimum Viable Trust Information)를 가장 먼저 제공해야 합니다.

| 우선순위 | 컴포넌트 명칭 | 핵심 역할 (What to Show) | 디자인/UX 전략 | 적용 KPI 연계 |
| :--- | :--- | :--- | :--- | :--- |
| **P1 (최우선)** | **시스템 상태 표시 컴포넌트 (Status Indicator)** | 현재 시스템의 **종합 안정성 상태**를 한눈에 제시. (🟢 정상, 🟡 주의, 🔴 위험) | 중앙 상단에 크고 명확한 색상 바와 핵심 지표(오류율/지연시간)를 카드 형태로 배치. | 오류율 ($\le 0.5\%$) 및 지연 시간 ($\pm 5\%$)의 **통합 상태**를 시각화. |
| **P2 (차순위)** | **KPI 추이 시각화 차트 (Trend Chart)** | 과거 일정 기간 동안 KPI(오류율, 지연시간)의 **변동 추이**를 시각적으로 비교. | 라인 차트(Line Chart)와 영역 차트(Area Chart)를 결합하여, 목표 범위(Threshold) 대비 현재 상태를 강조 표시. | 실시간 데이터 스트림을 기반으로 안정성 KPI의 장기적 추세를 제시. |
| **P3 (보조)** | **세부 로그/알림 모듈** | 시스템 오류 발생 시 **구체적인 원인과 해결책**을 제공. | 사용자가 클릭했을 때 관련 에러 코드, 타임스탬프, 그리고 Writer가 정의한 권장 조치(Actionable Insight)를 명시적으로 보여줌. | 특정 KPI 임계값 초과 시 즉각적으로 활성화되어 사용자 행동 유도. |

### 3. 최종 디자인 결정 (Design Finalization)

**P1 컴포넌트: 시스템 상태 표시 컴포넌트 상세 사양**

*   **컴포넌트 명:** `SystemStatusCard`
*   **레이아웃:** 가로형 카드 (Width: 100%)
*   **상태별 색상 매핑:**
    *   **정상 (OK):** 배경 `#d4edda` (Light Green), 텍스트/아이콘 `#28a745` (Green)
    *   **주의 (Warning):** 배경 `#fff3cd` (Light Yellow), 텍스트/아이콘 `#ffc107` (Amber) - *KPI가 임계값 근접 시 표시.*
    *   **위험 (Danger):** 배경 `#f8d7da` (Light Red), 텍스트/아이콘 `#dc3545` (Red) - *오류율이 $\le 0.5\%$를 초과하거나 지연 시간이 임계치를 벗어날 시.*
*   **핵심 정보 배치:**
    1.  **상태 아이콘 및 상태 텍스트 (가장 크게)**
    2.  **핵심 지표 표시 (Primary Metric):** 현재 오류율 (%) / 현재 지연시간 (ms)
    3.  **상태 변화 로그 링크:** 상세 알림 모듈(P3)로의 연결 버튼 포함.
