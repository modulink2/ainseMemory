## 📊 브랜드 비주얼 시스템 (핵심 가이드)

### 1. 브랜드 컬러 팔레트 (The Data-Driven Palette)

브랜드의 '데이터 기반 신뢰'라는 정체성을 반영하여, 차분하고 전문적이며 안정감을 주는 색상 조합을 제안합니다.

| 역할 | 색상명 | Hex Code | RGB | 용도 및 심리 효과 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary (신뢰/데이터)** | Deep Navy Blue | `#0A2342` | (10, 35, 66) | 시스템의 안정성, 전문성, 깊은 지식. 메인 배경 및 타이틀에 사용. |
| **Secondary (안정/분석)** | Slate Gray | `#6B7280` | (107, 114, 128) | 데이터 시각화, 차트 라인, 본문 텍스트 등 보조 정보에 사용. 중립적이고 분석적인 느낌 부여. |
| **Accent (경고/안전)** | System Green | `#38A169` | (56, 161, 105) | 성공, 안전장치(System Safety), 상승 예측 시 강조. 신뢰를 확인하는 포인트 색상. |
| **Background (배경)** | Off-White | `#F7F7F9` | (247, 247, 249) | 높은 가독성 확보. 데이터가 돋보이도록 깨끗한 배경 제공. |

**💡 컬러 적용 원칙:**
*   **Deep Navy Blue (`#0A2342`)**: 채널의 메인 타이틀, 주요 섹션 구분선에 사용하여 전문적이고 무게감 있는 분위기를 설정합니다.
*   **Slate Gray (`#6B7280`)**: 차트, 그래프 데이터 표시 및 상세 설명 텍스트에 사용합니다. 정보 전달의 정확성을 높입니다.
*   **System Green (`#38A169`)**: 시스템이 안전하게 작동하거나 긍정적인 결과(리스크 관리 성공)를 나타낼 때만 사용하여, 단순한 '돈'보다는 '안전성'을 강조하는 신뢰 색상으로 활용합니다.

### 2. 썸네일 디자인 가이드라인 (Thumbnail System)

제시된 영상 주제("시스템 안전장치", "$MDD_{limit}$")에 맞춰 시청자의 클릭을 유도하면서도 데이터 기반의 권위를 전달하는 세 가지 컨셉의 구체적인 레이아웃 및 요소를 정의합니다.

**공통 규칙:**
1.  **폰트:** 제목은 Bold한 Sans-Serif (예: Pretendard Bold), 보조 텍스트는 명확한 산세리프 (예: Noto Sans).
2.  **레이아웃:** 좌측 또는 우측에 인물(혹은 시스템 다이어그램) 배치, 중앙에 핵심 메시지 및 수치 강조.
3.  **톤앤매너:** 어둡고 전문적인 배경 위에 밝고 선명한 데이터 색상으로 대비를 극대화합니다.

#### 🖼️ 썸네일 컨셉 1: [The Safety Lock] (가장 강력 추천)

*   **핵심 메시지:** 시스템의 안전성과 통제력 강조.
*   **배경:** Deep Navy Blue (`#0A2342`) 또는 어두운 그레이 계열.
*   **시각 요소:** 중앙에 **자물쇠 아이콘** 또는 **방패 아이콘**을 크게 배치하고, 그 위에 `$MDD_{limit}` 수치를 굵고 선명한 System Green (`#38A169`)으로 강조합니다.
*   **텍스트:** "손실 제로 시스템", "$MDD_{limit}$ 완벽 방어" (최소화된 텍스트로 최대의 임팩트 전달)
*   **느낌:** 기술적이고, 보안에 중점을 둔 전문가 포지션 강조.

#### 🖼️ 썸네일 컨셉 2: [The Data Proof] (분석적 접근)

*   **핵심 메시지:** 예측이 아닌 데이터 분석의 우위 강조.
*   **배경:** Off-White (`#F7F7F9`) 또는 매우 밝은 그레이.
*   **시각 요소:** 좌측에 복잡한 주식 차트의 일부를 모자이크 처리하여 배치하고, 우측에는 깔끔한 **수학 공식이나 시스템 다이어그램**을 옅은 선으로 표현합니다.
*   **텍스트:** "예측 실패? 데이터가 답이다", "$MDD_{limit}$ 수학적 검증" (데이터의 논리적 흐름을 시각화)
*   **느낌:** 학술적이고, 데이터 분석에 기반한 깊은 통찰력을 어필.

#### 🖼️ 썸네일 컨셉 3: [The System Flow] (프로세스 강조)

*   **핵심 메시지:** 자동매매 시스템의 통합된 흐름(Flow) 강조.
*   **배경:** Deep Navy Blue (`#0A2342`).
*   **시각 요소:** 왼쪽에서 오른쪽으로 **단계별 프로세스 다이어그램**을 배치합니다. (예: 예측 모듈 $\rightarrow$ 리스크 헷지 $\rightarrow$ 안전장치 활성화) 각 단계는 System Green 색상으로 연결됩니다.
*   **텍스트:** "자동매매 시스템의 숨겨진 흐름", "안전장치 통합 과정" (프로세스의 투명성을 강조)
*   **느낌:** 시스템의 작동 원리 자체를 보여주어 기술적 깊이와 자동화의 신뢰도를 어필.
