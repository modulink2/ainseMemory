## 🖼️ 최종 비주얼 시스템 확정: 데이터 기반 신뢰 디자인

**핵심 컬러 정의:**
*   **Deep Navy Blue (#0A1931):** 시스템 안정성, 신뢰, 깊이 (기본 배경 및 주요 텍스트)
*   **System Green (#00C853):** 안전함, 성장, 통제 가능성 (핵심 지표, 성공 상태, Pro 기능 강조)
*   **Accent Gray (#B0B0B0):** 보조 정보, 비활성 요소

### 1. 썸네일 컨셉 3안 (Thumbnail Concepts)

각 썸네일은 '안정성'과 '차별점'을 명확히 하여 클릭률(CTR)을 높이는 데 초점을 맞춥니다.

#### **✨ 컨셉 A: 안전성 강조형 (Basic Shield 집중)**
*   **목표:** 시스템 안정성과 안전장치 제공을 최우선으로 어필.
*   **레이아웃:** 좌측 60%는 Deep Navy Blue 배경, 우측 40%는 System Green으로 강조되는 시각적 요소 배치.
*   **시각 요소:** 중앙에 큰 자물쇠 아이콘(🔒)을 배치하고, 그 주변에 '안정성' 관련 데이터 그래프를 미세하게 오버레이.
*   **텍스트 스타일:** 제목은 화이트/라이트 그레이, 핵심 키워드('안전장치')는 System Green 텍스트로 강조.
*   **톤앤매너:** 차분하고 신뢰감을 주며, '안심'을 주는 느낌.

#### **✨ 컨셉 B: 통제력 강조형 (Pro Hedger 집중)**
*   **목표:** 예측을 넘어선 '통제(Control)'와 '수익 극대화'라는 고급 가치 제시.
*   **레이아웃:** 좌측 50%는 Deep Navy Blue 배경, 우측 50%는 System Green의 명확한 대비를 활용하여 차이를 강조.
*   **시각 요소:** 중앙에 복잡하지만 정돈된 데이터 흐름(Flow) 또는 헷지 그래프를 배치하고, Pro 기능 영역에는 더 선명한 녹색으로 포인트를 줌.
*   **텍스트 스타일:** 'Pro' 티어는 System Green으로 처리하여 프리미엄 느낌 강조.
*   **톤앤매너:** 전문적이고 역동적이며, 고급 투자자에게 어필하는 느낌.

#### **✨ 컨셉 C: 비교 및 시스템 명확화형 (Tier Comparison)**
*   **목표:** Basic과 Pro의 기능 차이를 한눈에 비교하여 구독 결정 유도.
*   **레이아웃:** 화면을 세로 또는 가로로 분할하여 좌측(Basic)과 우측(Pro) 영역을 명확히 구분.
*   **시각 요소:** 두 섹션 사이에 수직선(Divider)을 Deep Navy Blue와 System Green의 그라데이션으로 처리. 각 티어별 핵심 기능 아이콘(안전장치 vs 헷지 전략)을 대비하여 배치.
*   **텍스트 스타일:** 명확한 비교 테이블 형식 사용.
*   **톤앤매너:** 데이터 기반의 명료함과 구조적 신뢰감을 극대화.

### 2. 핵심 UI 컴포넌트 디자인 초안: 구독 티어 차별화 (Subscription Tier Component)

사용자가 구독 옵션을 선택하는 화면에서 Basic과 Pro 간의 시스템 안정성 차이를 시각적으로 강조하는 컴포넌트를 제안합니다.

#### **📐 컴포넌트 명칭:** `Risk_Tier_Selector`

**디자인 목표:** 색상 대비와 아이콘 사용을 통해 기능적 우위를 직관적으로 전달합니다.

| 요소 | Basic Shield ($19/월) | Pro Hedger ($49/월) | 디자인 원칙 |
| :--- | :--- | :--- | :--- |
| **구독 박스 배경** | Deep Navy Blue (기본) | Darker Navy Blue with Subtle Green Gradient | 안정성과 프리미엄의 시각적 차이. |
| **안정성 아이콘** | 🛡️ **Basic Shield Icon** (단순 방패 모양, System Green 테두리) | 🛡️ **Pro Hedger Icon** (복잡한 네트워크/헷지 형태, 밝은 System Green 채움) | 기능의 깊이와 복잡성을 시각화. |
| **핵심 가치 문구** | "시스템 안전장치" (안정성 확보) | "리스크 통제 엔진" (수익 극대화) | 제공하는 핵심 가치의 차이를 명확히 구분. |
| **가격 표시** | $19 /월 (White Text) | $49 /월 (Bright System Green Text) | Pro 티어를 가장 눈에 띄는 색상으로 강조하여 프리미엄 포지셔닝. |

**레이아웃 시각화 예시 (Conceptual Layout):**

```mermaid
graph TD
    subgraph Risk_Tier_Selector
        A[Basic Shield ($19/월)] --> B{🛡️ 안전장치}
        C[Pro Hedger ($49/월)] --> D{⚙️ 리스크 통제 엔진}
    end
    style A fill:#0A1931,stroke:#00C853,stroke-width:2px
    style C fill:#0A1931,stroke:#00C853,stroke-width:3px
```

**디자인 해석:**
Basic Shield는 시스템 안정성이라는 '방어막'에 초점을 맞추고, Pro Hedger는 리스크를 능동적으로 관리하고 수익을 극대화하는 '엔진(Engine)'에 초점을 맞춥니다. System Green의 사용 정도와 아이콘의 복잡도를 통해 기능적 우위를 즉각적으로 인지하게 만듭니다.
