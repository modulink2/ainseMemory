CEO의 지시대로, 최종 확정된 PTI 기반 데이터 포맷(JSON 스키마)과 개발된 알고리즘 예측 결과가 대시보드에 정확히 반영되는지 시각화 로직의 일치성을 검토하고, 시스템 안정성 지수(PTI)를 최우선으로 강조하는 UI 구현을 위한 상세 명세를 확정하겠습니다.

이 명세는 **시스템 안정성(PTI)**을 핵심 축으로 삼아, 사용자가 리스크 대비 수익률보다 '안전한 예측'에 집중하도록 유도하는 시각적 계층 구조를 정의합니다.

# 🖥️ 대시보드 UI/UX 상세 명세 (PTI 중심)

## 1. 디자인 철학 및 핵심 원칙
**핵심 메시지:** "수익률(ROI)보다 시스템 안정성(PTI)이 우선이다."
**색상 체계 재확인:**
*   **Primary Color (안정성):** Deep Navy Blue (시스템 안정성, 신뢰성 강조)
*   **Accent Color (행동/성과):** System Green (긍정적 예측, 안전한 성장 지표)
*   **Warning/Risk Color (경고):** Amber/Orange (PTI 위험 수준 시각화)
*   **Danger Color (위험):** Red (시스템 불안정 또는 높은 리스크 경고)

## 2. 데이터 시각화 로직 명세 (JSON 스키마 반영)
모든 지표는 반드시 **PTI(System Stability Index)**에 의해 색상과 위치가 결정되어야 합니다.

| 데이터 필드 | 측정 기준 | 시각화 방법 | PTI 연동 방식 | 강조 우선순위 |
| :--- | :--- | :--- | :--- | :--- |
| **PTI (System Stability Index)** | 시스템 안정성 지수 (0~100) | 대시보드 최상단 KPI 카드, 전체 배경 톤 결정 | **최우선**. 색상(Deep Navy Blue $\rightarrow$ Green/Amber/Red 스케일링). | ⭐⭐⭐ |
| **예측 성공률** | 알고리즘 예측의 실제 달성 비율 | 원형 차트 또는 게이지 (%) | PTI가 높을수록 이 지표의 신뢰도 증대. | ⭐⭐ |
| **예측 결과 (상승/급등/강성)** | 알고리즘이 예측한 주식 카테고리 | 바 차트 (Bar Chart) | 각 예측 값 옆에 해당 예측의 PTI 점수를 마커로 표시. | ⭐⭐⭐ |
| **리스크 레벨** | 예상되는 시스템 리스크 지수 | 히트맵 또는 상태 불빛(Indicator Light) | PTI가 특정 임계값 이하일 경우, 리스크 색상(Amber/Red)으로 즉시 변경. | ⭐⭐ |

## 3. 티어별 UI 차별화 (Basic vs Pro)

| 요소 | Basic Tier (안정성 우선) | Pro Tier (성과 및 관리 우선) |
| :--- | :--- | :--- |
| **PTI 시각화** | 전체 대시보드 배경은 Deep Navy Blue 기반. PTI는 주로 KPI 카드에 명확한 수치와 색상으로 표시. | **시스템 안정성 지수(PTI)**가 메인 레이아웃의 핵심 축이 됨. PTI 변화에 따라 메인 영역의 밝기나 구도가 동적으로 변화함. |
| **데이터 밀도** | 핵심 예측 결과 (3가지 카테고리)만 강조. 데이터는 간결하게 제시하여 오해를 줄임. | 모든 알고리즘 예측(상승/급등/강성) 및 상세 리스크 레벨을 상세 그래프로 제공. 데이터 밀도를 높여 심층 분석 지원. |
| **액션 버튼** | 단순 조회 및 알림 기능에 집중. | 자동 매매 전략 연동, 리스크 조정 시뮬레이션 등 **관리 및 액션 중심의 인터랙티브 버튼** 강화. |

## 4. 최종 레이아웃 제안 (DashboardLayout 기반)

1.  **상단 바 (Nav/Header):** 로고와 함께 현재 시스템 안정성 지수(PTI)를 가장 크게 표시하고, PTI 색상을 배경으로 사용한다.
2.  **사이드바 (Navigation):** Portfolio Kit의 구조를 따르되, '시스템 안정성 대시보드'가 최상위 메뉴로 위치하여 항상 접근 가능하게 한다.
3.  **메인 영역 (Main Content):**
    *   **상단 패널:** PTI 현황 요약 카드 (PTI, ROI, 리스크 레벨).
    *   **중앙 섹션:** 알고리즘 예측 결과 시각화 (Bar Chart 및 히트맵). **여기서 모든 시각적 요소는 PTI의 색상 스케일에 종속되어야 한다.**
    *   **하단 섹션:** 상세 데이터 테이블 및 알림 로그.
