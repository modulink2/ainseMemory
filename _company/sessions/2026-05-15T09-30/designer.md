확정된 PTI 중심의 UI/UX 명세를 바탕으로, 시스템 안정성 지수(PTI)를 가장 강조하는 대시보드 와이어프레임 구성을 즉시 시작하여 프로토타입 개발을 위한 기반을 마련하겠습니다. 핵심은 **데이터 기반 신뢰감**을 시각화하고, **시스템 안정성 지수(PTI)**가 사용자에게 즉각적으로 인지되도록 구조를 설계하는 것입니다.

---

## 📊 대시보드 와이어프레임 구상 (PTI 중심)

### 1. 디자인 시스템 요약
*   **핵심 컬러 팔레트:**
    *   **Primary (신뢰/안정):** Deep Navy Blue (`#0A1931`) - 배경, 주요 구조 요소
    *   **Accent (성장/안전):** System Green (`#38A169`) - PTI 수치, 긍정적 지표 강조, 성공 상태
    *   **Neutral:** Light Gray (`#F7FAFC`), Dark Text (`#1A202C`)

### 2. 레이아웃 구조 (DashboardLayout 기반)
대시보드는 `DashboardLayout` (사이드바 + 탑바 + 메인 영역 통합) 구조를 따르며, PTI 강조를 위해 왼쪽 또는 상단에 핵심 지표를 배치합니다.

| 영역 | 구성 요소 | 주요 목적 및 디자인 적용 |
| :--- | :--- | :--- |
| **A. 사이드바 (Navigation)** | 로고, 메뉴 (대시보드, 알고리즘 상세, 리스크 관리, 설정) | Deep Navy Blue 배경에 System Green 하이라이트 텍스트로 활성 상태 표시. |
| **B. 탑 바 (Header)** | 사용자 정보, 시스템 상태 표시 (실시간 알림), 전체 PTI 요약 지표 | 현재 세션의 전반적인 안정성 상태(PTI 레벨)를 한눈에 파악 가능하도록 배치. |
| **C. 메인 영역 (Main Content Area)** | **PTI 핵심 대시보드** 및 상세 데이터 시각화 | 사용자가 가장 집중해야 할 PTI와 ROI 관계를 중심으로 구성. |

### 3. 핵심 위젯 상세 설계: PTI 강조 전략

**목표:** 시스템 안정성 지수(PTI)가 모든 정보의 최상위에 위치하도록 한다.

#### 3.1. 메인 KPI 카드 (Top Section - PTI Focus)
*   **위치:** 화면 중앙 상단, 가장 눈에 띄는 영역 (Deep Navy Blue 배경).
*   **구성:** 세 개의 큰 카드 형태로 구성하여 **PTI, ROI, 시스템 안전 모드 상태**를 시각적으로 분리.
    1.  **시스템 안정성 지수 (PTI):** 가장 큰 폰트로 표시. 현재 PTI 값과 이전 대비 변화율(%)을 System Green으로 강조.
    2.  **수익률 (ROI):** PTI와 연동된 실시간 수익률 표시.
    3.  **안전 모드 상태:** 시스템 안전 모드 활성화 여부를 명확한 색상(예: 노란색/빨간색 경고)으로 표시.

#### 3.2. 데이터 시각화 섹션 (Visualization Section)
*   **목적:** 알고리즘 결과와 PTI의 상관관계를 보여줌.
*   **구성 요소:**
    *   **PTI 추이 그래프:** 시간 흐름에 따른 PTI 변화를 Line Chart로 표시. (Deep Navy Blue 선, System Green 강조).
    *   **알고리즘 성과 매트릭스:** 상승/급등/강성 예측 결과와 해당 시점의 PTI 점수를 연결하는 히트맵 또는 바 차트. (예: '상승 예측' 시점에서 PTI가 높거나 낮은 패턴 시각화).

#### 3.3. 리스크 및 상세 모듈 섹션 (Risk & Detail Modules)
*   **구성:** 하단 영역에 배치하여 상세 정보 접근성을 확보.
    *   **리스크 관리 패널:** 현재 포트폴리오의 리스크 노출도(PTI 기반)와 헤징 상태를 요약하여 표시.
    *   **알고리즘 로그:** API 응답 데이터 및 PTI 계산 과정에 대한 상세 로그를 열람할 수 있는 탭 구조 제공.

### 4. 디자인 가이드라인 적용 (Color & Typography)

*   **Deep Navy Blue (`#0A1931`):** 모든 배경, 주요 컨테이너 영역에 사용하여 전문적이고 안정적인 느낌을 부여합니다.
*   **System Green (`#38A169`):** **PTI 점수가 75% 이상일 때**, 또는 시스템이 '안정 상태'일 때 사용하며 신뢰와 성장의 시각적 코드로 활용합니다.
*   **폰트:** 데이터의 명확성을 위해 Sans-serif 계열 (예: Pretendard 또는 Inter)을 사용하며, 수치(PTI, ROI)는 굵고 큰 서체로 처리하여 즉각적인 인지도를 높입니다.
