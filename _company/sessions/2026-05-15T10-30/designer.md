# 💎 갓더주식: 신뢰 기반 비주얼 디자인 가이드라인 (PTI 중심)

## 1. 브랜드 컬러 및 타이포그래피 정의

| 요소 | 색상 코드 | 역할/의미 | 적용 범위 |
| :--- | :--- | :--- | :--- |
| **Primary Color (안정성)** | `#0A1931` (Deep Navy Blue) | 신뢰, 깊이, 시스템 안정성(PTI)의 기반. 배경 및 주요 구조에 사용. | 대시보드 배경, 네비게이션 바, 핵심 데이터 영역 테두리. |
| **Accent Color (수익/안전)** | `#00C853` (System Green) | 성장, 안정적인 수익(ROI), 시스템의 긍정적 상태(PTI > 기준치). | PTI 지표, 성공 및 상승 표시, 핵심 CTA 버튼, 긍정적 피드백. |
| **Neutral Color (데이터)** | `#F4F4F8` (Light Gray/Off-White) | 정보의 가독성을 높이고 데이터 영역을 분리하여 명확성 확보. | 카드 배경, 차트 영역, 텍스트 본문. |
| **Typography (폰트)** | Pretendard 또는 Noto Sans KR | 데이터 기반의 전문성과 깔끔함을 제공하는 산세리프 계열. | 모든 UI 텍스트 및 그래프 레이블. |

## 2. 대시보드 UI/UX 핵심 원칙: PTI 시각화 전략

대시보드의 최우선 목표는 사용자가 **"시스템이 안전하고 예측 가능하다"**고 즉각적으로 인식하게 만드는 것입니다. 이는 ROI(수익)보다 PTI(안정성)를 우선 배치하는 것을 의미합니다.

### A. 지표 계층 구조 (Hierarchy of Information)

1.  **최상단 (The Trust Bar):** 대시보드 상단에 **PTI 점수**를 가장 크고 눈에 띄게 배치합니다.
    *   **시각화:** PTI 수치 옆에 작은 게이지 또는 색상 바(System Green/Red)를 즉시 표시하여 현재 시스템의 안정 상태를 한눈에 파악하게 합니다.
2.  **중앙 (The Core Metrics):** ROI와 같은 수익 지표를 배치하되, 반드시 **PTI와의 관계**를 함께 보여줍니다.
    *   **연결 고리:** 각 ROI 수치 옆에는 해당 수익이 달성된 시스템의 PTI 수준을 괄호나 보조 지표로 표시합니다. (예: `ROI: +15% (PTI: 92)`)
3.  **하단 (The Detail & Action):** 상세 데이터와 예측 알고리즘 결과, 그리고 다음 행동(CTA)을 배치합니다.

### B. 색상 사용 규칙 (Color Application Rules)

*   **안정성 표시:** PTI가 기준치 이상일 경우 **System Green**으로 강조하여 '안전함'을 시각적으로 보증합니다.
*   **위험 경고:** PTI가 특정 임계치(예: 70점 미만) 이하로 떨어질 경우, 해당 지표와 관련 섹션 전체에 **Deep Navy Blue의 어두운 톤**이나 대비되는 **Red Accent**를 사용하여 즉각적인 주의를 환기시킵니다.
*   **데이터 비교:** ROI 변화나 예측 결과는 System Green을 사용하여 긍정적 추세를 강조하고, 위험 요소(변동성)는 회색 계열로 중립적으로 처리합니다.

### C. 데이터 시각화 (Visualization Style)

*   **차트 스타일:** 모든 핵심 차트는 선형 그래프(Line Chart)를 기본으로 하되, **시스템 안정성을 나타내는 지표(PTI)**는 반드시 영역 차트(Area Chart)나 게이지 형태로 표현하여 '범위'와 '안정성'을 강조합니다.
*   **레이아웃:** 데이터는 왼쪽/상단에 배치하고, 시스템의 핵심 상태 표시(PTI)는 중앙 상단에 위치시켜 **정보의 흐름이 논리적이고 안정적으로 보이도록** 구성합니다.

## 3. 마케팅 및 콘텐츠 적용 가이드라인 (Trust Messaging)

디자인 가이드라인은 대시보드뿐만 아니라 외부 콘텐츠에도 일관되게 적용되어야 합니다.

### A. 핵심 슬로건 구조 (The Trust Formula)

모든 마케팅 메시지는 다음의 논리적 흐름을 따라야 합니다.

$$\text{시스템 안정성 (PTI)} \rightarrow \text{수익 예측 (ROI)} \rightarrow \text{사용자 신뢰}$$

### B. 썸네일 디자인 가이드라인 (Thumbnail Design)

썸네일은 '데이터 기반의 확신'을 시각적으로 전달해야 합니다.

1.  **배경:** Deep Navy Blue 또는 어두운 톤을 사용하여 전문성과 깊이를 확보합니다.
2.  **핵심 요소:** **System Green** 색상을 활용하여 가장 중요한 지표(예: PTI 점수)를 강조하고, 그 옆에 '안정성'이나 '검증됨'과 같은 신뢰 관련 키워드를 배치합니다.
3.  **시각적 대비:** 불안정한 차트 이미지(약간 흐릿하거나 빨간색 톤)와 안정적인 대시보드 화면(선명한 System Green 톤)을 대비시켜, **"우리가 제공하는 것은 불안이 아닌 확실성"**이라는 메시지를 간접적으로 전달합니다.
