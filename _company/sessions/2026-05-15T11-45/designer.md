CEO의 지시에 따라 현빈과 코다리가 정의한 PTI 중심의 UI/UX 명세를 바탕으로 시스템 안정성 지수(PTI)가 가장 강조되는 대시보드 프로토타입 개발을 즉시 시작하며, Deep Navy Blue와 System Green 색상을 적용하여 디자인 가이드를 구체화합니다.

**1. 브랜드 및 컬러 시스템 확정 (Visual System)**

시스템 안정성과 신뢰를 시각적으로 표현하는 핵심 컬러 팔레트와 타이포그래피 스타일을 최종 확정합니다.

*   **Primary Color (신뢰/안정성):** Deep Navy Blue
    *   HEX: `#0A1931` (최고 안정성을 상징)
    *   Usage: 배경, 주요 프레임, 핵심 데이터 영역
*   **Accent Color (성장/안정화):** System Green
    *   HEX: `#16A34A` (시스템 안정성과 긍정적 상태를 상징)
    *   Usage: PTI 지수 표시, 성공/강성 예측 결과, 핵심 CTA 버튼
*   **Neutral Color (정보 계층):** Light Gray / White
    *   HEX: `#F5F5F5` (배경 영역), `#FFFFFF` (카드 배경)
*   **Typography:** 데이터 기반의 신뢰감을 위해 산세리프 계열을 사용합니다.
    *   Font Family: Pretendard (또는 Inter, 시스템 환경에 따라 선택)
    *   Heading Font: Bold, Strong weight for emphasis on KPIs.

**2. 대시보드 프로토타입 구성 (Dashboard Prototype Concept)**

PTI를 최우선으로 배치하여 사용자가 즉각적으로 시스템의 안정 상태를 파악할 수 있도록 설계합니다.

| 영역 | 내용 | 시각적 강조 요소 | 색상 적용 |
| :--- | :--- | :--- | :--- |
| **A. 상단 내비게이션 (Header)** | 로고, 사용자 정보, 알림 기능 | 최소화, 명확성 | Deep Navy Blue 배경 |
| **B. 핵심 KPI 요약 (PTI Focus Area)** | **시스템 안정성 지수 (PTI) 대시보드** | 가장 크고 중앙 배치 | **System Green** 배경 강조 |
| **C. 리스크 및 수익률 섹션** | 실시간 예측 결과 (상승/급등/강성), ROI 연동 데이터 | 그래프, 시각적 추세 표시 | Deep Navy Blue 배경에 System Green 라인 |
| **D. 상세 모듈 (Detail Modules)** | 포트폴리오 현황, 리스크 헷지 상태 등 | 카드(Card) 형태로 분리하여 정보 계층화 | White/Light Gray 배경 |

**3. 디자인 가이드라인 (Visual System Guideline)**

*   **PTI 시각화:** PTI 값은 중앙에 매우 크게 표시하며, 현재 상태에 따라 색상을 동적으로 변화시킵니다.
    *   `PTI > 90%`: System Green (안정)
    *   `60% < PTI <= 90%`: Deep Navy Blue (주의/모니터링 필요)
    *   `PTI <= 60%`: Dark Red (위험/경고)

*   **데이터 시각화:** 모든 차트와 그래프는 Line Chart 또는 Gauge Meter 형태를 기본으로 하며, 데이터의 추이(Trend)와 현재 상태(Status)가 명확히 드러나도록 설계합니다.

**4. 썸네일 컨셉 3안 (Thumbnail Concepts)**

시스템 안정성과 수익률 연결이라는 핵심 메시지를 중심으로 세 가지 컨셉을 제안합니다.

| 컨셉 | 핵심 메시지 | 시각적 강조점 | 추천 컬러 조합 |
| :--- | :--- | :--- | :--- |
| **컨셉 1: 시스템 신뢰 (The Stability)** | "데이터 기반의 완벽한 안정성" | 대형 PTI 수치와 녹색(Green)의 명확한 대비 | Deep Navy Blue & System Green |
| **컨셉 2: 수익 연결 (The Profit Link)** | "안정성이 곧 수익이다. ROI 극대화" | 두 개의 KPI(PTI, ROI)를 양방향 화살표로 연결하여 상호 의존성 강조 | Deep Navy Blue & Gold/Green Accent |
| **컨셉 3: 예측 기반 (The Predictive Edge)** | "알고리즘이 제시하는 확실한 미래" | 시스템 안정성 지수가 그래프의 최고점(Peak)을 형성하며 상승하는 모습 시각화 | Deep Navy Blue & System Green (상승 곡선 강조) |
