# 📊 최종 대시보드 UI/UX 디자인 가이드라인: 안정성이 곧 수익

## 1. 디자인 철학 (Design Philosophy)
**핵심 메시지:** "안정성(Stability, PTI)이 곧 수익(Profit, ROI)이다."
**목표:** 사용자가 복잡한 수치에서 시스템의 안정성이 직접적인 수익성에 미치는 인과관계를 직관적으로 이해하고 신뢰하도록 한다.
**톤앤매너:** 데이터 기반, 신뢰감, 명확성, 통제감(Control).

## 2. 컬러 팔레트 (Color Palette)
시스템 안정성과 수익성을 시각적으로 구분하기 위해 두 가지 핵심 색상을 사용합니다.

*   **Primary Color (안정성/Stability):** `Deep Navy Blue` (신뢰, 안정성, 시스템 상태 강조) - `#001F3F`
*   **Accent Color (수익/Profit):** `System Green` (성과, 성장, 긍정적 수익률 강조) - `#39FF14`
*   **Background:** Dark Mode 기반의 어두운 배경 (`#121212` 또는 유사한 짙은 회색 계열)
*   **Neutral Text/Data:** 밝은 회색 계열 (정보 전달의 명료성 확보)

## 3. 레이아웃 구조 (Layout Structure: DashboardKit 기반)
기존 `DashboardLayout`을 확장하여 핵심 지표를 최상단에 배치하고, 나머지 영역은 세부 분석 및 리스크 관리에 할당합니다.

**A. 상단 바 (Header/Navigation)**
*   **좌측:** 로고 (`갓더주식`), 좌측 메뉴 (Portfolio Kit의 Nav 참조)
*   **우측:** 사용자 프로필, 알림, 시스템 상태 표시 (PTI 현황 요약)

**B. 핵심 KPI 대시보드 영역 (The Stability-Profit Core)**
이 영역은 화면 상단에 가장 크게 배치하여 사용자가 로그인 시 즉시 시스템의 '현재 상태'를 파악하게 합니다.

1.  **시스템 안정성 지수 (PTI) 위젯:**
    *   가장 큰 폰트로 표시.
    *   중앙에 **Deep Navy Blue** 배경 위에 **System Green** 색상의 게이지 또는 바 차트를 배치하여 PTI의 현재 레벨을 시각화합니다.
    *   표시 텍스트: "현재 시스템 안정성 지수 (PTI): [XX.X]"
2.  **수익률 지표 (ROI) 위젯:**
    *   PTI 위젯 바로 아래에 배치.
    *   `ROI` 값을 크게 표시하고, 이 값이 PTI의 상태와 어떻게 연관되는지를 설명하는 텍스트(예: "안정성 기반 예상 수익률")를 함께 명시합니다.
3.  **상태 요약 박스 (Status Summary Box):**
    *   PTI/ROI에 따른 최종 판단을 제공하는 핵심 영역.
        *   **안정성 레벨 표시:** PTI 수치에 따라 `Basic Shield` 또는 `Pro Hedger` 상태를 명확히 표시합니다.
        *   **추천 액션:** 현재 시스템 상태에 맞는 다음 행동(예: "안정성 강화 필요", "수익률 최적화")을 제안하는 CTA 버튼 포함.

**C. 심층 분석 영역 (Deep Dive Analysis)**
이 영역은 `business_spec.md`의 세부 데이터를 분석하고 예측 결과를 시각화합니다.

1.  **PTI vs ROI 관계 시각화 차트:**
    *   **이중 축 그래프 (Dual-Axis Chart):** X축은 시간(Time) 또는 알고리즘 실행 주기, Y축은 PTI와 ROI를 동시에 표시하여 두 지표의 상관관계를 선 그래프 등으로 표현합니다.
    *   **핵심 시각화:** 안정성(PTI)의 변화가 수익률(ROI)에 미치는 영향을 **색상 그라데이션** 또는 **영역 채우기**로 표현합니다. (예: PTI가 낮을 때 ROI 영역은 붉은색/노란색, PTI가 높을 때 ROI 영역은 녹색으로 명확히 구분)
2.  **알고리즘 예측 결과 상세:**
    *   상승예측, 급등예측, 현재강성주 파악 결과를 별도의 카드 형태로 제공합니다. 각 예측 항목별로 **신뢰도 점수**와 함께 PTI와의 연관성을 간략히 설명합니다.

## 4. 디자인 가이드라인 (Visual Guidelines)

| 요소 | 적용 규칙 | 상세 지침 |
| :--- | :--- | :--- |
| **데이터 시각화** | **게이지 및 영역 차트 우선** | 단순한 막대 그래프보다 PTI/ROI의 '상태'를 나타내는 게이지와 색상 기반 영역 채우기를 사용하여 즉각적인 인식을 유도한다. |
| **색상 사용** | **Deep Navy Blue & System Green** | Deep Navy Blue는 시스템, 안정성, 배경에, System Green은 성과, 긍정적 수익률, 성공 지표에만 한정하여 사용한다. 경고나 위험 신호에는 Amber/Red를 제한적으로 사용한다. |
| **정보 계층** | **PTI 최우선 강조** | 모든 시각적 요소에서 PTI 수치와 그 해석이 ROI 수치보다 더 높은 위계를 갖도록 디자인하며, PTI가 핵심 변수임을 끊임없이 상기시킨다. |
| **데이터 흐름** | **인과관계 명시** | 각 예측 결과 카드 아래에는 "이 예측은 현재 시스템 안정성 지수(PTI) [XX]%를 기준으로 산출되었습니다"와 같이 인과관계를 텍스트로 반드시 명시한다. |
