## 1. 대시보드 와이어프레임 초안 (Dashboard Wireframe Draft)

**목표:** 시스템 안정성 지수(PTI)를 최우선으로 배치하여 사용자가 플랫폼의 가장 중요한 가치(안정성)를 즉시 인식하도록 설계합니다.

**핵심 원칙:**
*   **최우선 KPI:** PTI (시스템 안정성 지수)가 화면 최상단 중앙에 위치해야 합니다.
*   **색상 활용:** Deep Navy Blue를 배경 및 주요 구조에 사용하고, System Green을 수익/안정성 관련 지표에 집중적으로 사용하여 시각적 우선순위를 부여합니다.

### A. 레이아웃 구성 (Layout Structure)

| 영역 | 내용 | 디자인 요소 및 색상 적용 | 비고 |
| :--- | :--- | :--- | :--- |
| **Header** | 로고, 사용자 정보, 안전 모드 버튼 | Deep Navy Blue 배경, 텍스트 White/Light Gray | 안정적인 느낌 강조 |
| **Top KPI Block (최우선)** | **PTI (System Stability Index)** | **가장 큰 폰트 크기**, System Green으로 강조. 그래프 또는 게이지 형태로 표시. | **시스템의 심장부** |
| **Secondary KPI Block** | ROI (수익률), 현재 포지션, 위험 지표 | Navy Blue 배경에 시스템 정보(데이터 기반)를 명확히 표시 | 데이터 비교 영역 |
| **Risk/Safety Control** | 자동매매 허용/정지 스위치 (`system_safe_mode`) | 빨간색 경고/녹색 활성화 상태로 즉각적인 조작 가능하게 배치. | **안전 모드 최상단 배치** |
| **Main Visualization Area** | 실시간 포트폴리오 및 지표 그래프 | Navy Blue 배경 위에서 데이터를 명확히 비교할 수 있는 차트(Line, Bar) 사용. | PTI와 ROI의 상관관계 시각화 |
| **Footer** | 시스템 상태 표시, 최종 업데이트 시간 | 작은 글씨로 Deep Navy Blue 배경에 배치. | 안정적인 마무리 |

### B. 핵심 디자인 요소 상세 명세 (Key Element Specification)

1.  **PTI 강조:**
    *   **위치:** 대시보드 상단 중앙 섹션의 가장 큰 영역을 차지합니다.
    *   **디자인:** 원형 또는 깔끔한 게이지(Gauge) 형태로 표시하며, **System Green** 색상을 주 컬러로 사용합니다.
    *   **텍스트:** "PTI: [현재 값]" (가장 크고 굵게).

2.  **색상 팔레트 적용:**
    *   **Primary Color:** Deep Navy Blue (`#0A1931` 또는 유사한 깊은 네이비) - 배경, 주요 패널.
    *   **Accent Color:** System Green (`#00FF7F` 또는 밝고 선명한 녹색 계열) - PTI 지표, 긍정적 수익 지표(ROI), 활성화된 안전 모드 상태.
    *   **Warning/Alert:** Red (경고) 및 Amber (주의)를 데이터 무결성 실패나 위험 임계치 초과 시에만 사용합니다.

3.  **폰트 스타일:**
    *   모든 텍스트는 **산세리프 계열** (예: Pretendard, Noto Sans KR 등)을 사용하여 데이터의 명료성과 전문성을 극대화합니다.

## 2. 유튜브 인트로 화면 와이어프레임 초안 (YouTube Intro Screen Wireframe Draft)

**목표:** 시청자에게 '갓더주식'이 단순 예측 시스템이 아닌, **'리스크 제어형 스마트 투자 플랫폼'**임을 즉시 전달하여 신뢰감(Trust)을 구축합니다.

**핵심 원칙:**
*   **톤앤매너:** 데이터 기반의 안정성과 통제감을 시각적으로 표현합니다. (Deep Navy Blue & System Green 활용)
*   **메시지:** '수익'보다 '안정성'이 우선임을 강조합니다.

### A. 레이아웃 구성 (Layout Structure)

| 영역 | 내용 | 디자인 요소 및 색상 적용 | 비고 |
| :--- | :--- | :--- | :--- |
| **Opening Hook (0-5초)** | 핵심 질문/선언 문구 | 화면 중앙에 굵고 강력하게 배치. Deep Navy Blue 배경 위에 System Green 텍스트로 강조. | 시청자 즉각적 후킹 |
| **System Trust Visual** | 안정성(PTI)의 시각화 | 복잡한 데이터 대신, PTI 지표가 안정적으로 유지되는 그래프나 시스템 아키텍처를 미니멀하게 표현. | '안정성이 곧 수익' 논리 시각화 |
| **Platform Introduction** | 플랫폼 이름 및 핵심 가치 | 갓더주식 로고와 함께 "리스크 제어형 스마트 투자 플랫폼" 문구 삽입. | 브랜드 정체성 확립 |
| **Call to Action (CTA)** | 구독 유도 또는 다음 단계 안내 | 깔끔한 Navy Blue 배경 위에 System Green 버튼으로 배치. | 명확한 행동 유도 |

### B. 핵심 시각 디자인 상세 명세 (Visual Specification)

1.  **배경:**
    *   전체 화면은 **Deep Navy Blue**로 통일하여 깊이감과 전문성을 부여합니다.

2.  **핵심 비주얼 요소 (PTI 강조):**
    *   화면 중앙에 **PTI 지표를 중심으로 한 심플한 원형 게이지 또는 안정성 바(Bar)**를 배치합니다. 이 게이지는 현재 시스템의 '안정성 레벨'을 나타내며, **System Green**으로 채워져야 합니다. (예: 95% 이상일 경우 녹색이 매우 밝게 빛남)

3.  **타이포그래피 및 메시지:**
    *   **헤드라인 예시:** "수익률보다 중요한 것. 시스템 안정성(PTI)" 또는 "안정성이 곧 수익이다."
    *   폰트는 명료하고 전문적인 느낌을 주는 산세리프체를 사용합니다.

4.  **전체 톤앤매너 요약:**
    *   화면은 차분하고, 데이터에 기반하며, **신뢰감(Trust)**이 최우선적으로 느껴져야 합니다. 과도한 움직임이나 화려한 색상 사용을 지양하고, 깊은 네이비와 선명한 녹색의 대비를 통해 전문성과 안정성을 동시에 전달합니다.
