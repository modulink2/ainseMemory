# 🚀 갓더주식 비주얼 시스템 최종 가이드라인 보고서

본 가이드라인은 **'리스크 제어형 스마트 투자 플랫폼'**이라는 회사 정체성(데이터 기반, 시스템 안정성 우선)을 시각적으로 구현하는 데 중점을 둡니다. 핵심 컬러와 타이포그래피를 일관되게 적용하여 사용자에게 **신뢰감과 통제감**을 즉각적으로 전달하는 것을 목표로 합니다.

## 1. 브랜드 핵심 디자인 요소 (Visual System Core)

| 요소 | 정의 | 사용 목적 | 코드/규칙 |
| :--- | :--- | :--- | :--- |
| **Primary Color (신뢰)** | Deep Navy Blue (딥 네이비 블루) | 플랫폼의 안정성, 전문성, 신뢰도 강조 | `#0A1931` |
| **Accent Color (안정/성장)** | System Green (시스템 그린) | 시스템 안정성(PTI), 긍정적 결과, 안전 모드 상태 시각화 | `#10B981` |
| **Secondary Color** | Light Gray / Off-White | 배경 및 정보 영역 분리, 데이터의 명확성 확보 | `#F7FAFC` |
| **Typography (주요)** | Noto Sans KR / Pretendard | 데이터 기반의 깔끔함과 가독성 극대화 | 굵기(Weight): Bold (700) for Titles, Regular (400) for Body |

## 2. 썸네일 컨셉 표준화 및 적용 방안

제작된 3가지 썸네일 초안(`youtube_thumbnail_*`)을 기반으로, **시스템 안정성 강조**라는 핵심 메시지를 극대화하는 최종 적용 방안을 확정합니다.

### A. YouTube 썸네일 가이드라인 (핵심: 안전성 시각화)

**목표:** 시스템 안정성(PTI)이 수익률보다 우선함을 직관적으로 전달하여 클릭 유도.

| 요소 | 디자인 규칙 | 적용 상세 내용 |
| :--- | :--- | :--- |
| **배경 컬러** | Deep Navy Blue (`#0A1931`) 또는 Dark Gray 계열 | 전문적이고 진중한 분위기 조성 |
| **핵심 강조** | System Green (`#10B981`) | PTI 수치, 안전 모드 상태 표시 등 핵심 KPI에만 사용. (정보 과부하 방지) |
| **타이포그래피** | 화이트 또는 밝은 그레이 | 대비 극대화하여 정보의 명료성 확보 |
| **레이아웃 구조** | 좌측 1/3: 시스템 안정성 지표($PTI$) 시각화 (System Green 강조) / 우측 2/3: 핵심 메시지 및 ROI 비교 | 데이터 기반 논리를 시각적으로 즉시 전달 |
| **레퍼런스 파일 활용** | `youtube_thumbnail_SystemStability_Final.png`의 구도를 유지하되, System Green의 사용 빈도를 높여 안정성 지표를 중앙에 배치하도록 수정. |

### B. 소셜 미디어 배너 가이드라인 (핵심: 브랜드 일관성)

제작된 2가지 배너(`banner_SystemStability_Guide.png`, `banner_SystemStability_CTA.png`)는 플랫폼의 **신뢰 기반**을 전달하는 데 사용됩니다.

1.  **`banner_SystemStability_Guide.png` (가이드/정보 제공용):**
    *   **톤앤매너:** 설명적이고 차분함.
    *   **컬러 분배:** Deep Navy Blue를 주 배경으로 사용하고, System Green을 섹션 구분선(Divider) 및 중요 데이터 강조에만 제한적으로 사용하여 전문성을 유지합니다.
    *   **텍스트 배치:** 정보의 위계를 명확히 하기 위해 폰트 크기를 체계적으로 조정합니다 (헤드라인 > 본문 > 상세 설명).

2.  **`banner_SystemStability_CTA.png` (행동 유도용):**
    *   **톤앤매너:** 행동 촉구(Call to Action) 중심.
    *   **컬러 분배:** 시스템 그린을 CTA 버튼 영역에 집중 배치하여 사용자의 시선을 즉시 유도합니다. Deep Navy Blue는 배경으로 사용하여 대비를 높입니다.
    *   **레이아웃 구조:** 중앙에 명확한 행동 버튼(CTA)을 System Green 박스로 강조하고, 그 위에 '안정성 확보'와 같은 핵심 문구를 배치하여 신뢰감을 기반으로 클릭을 유도합니다.

## 3. 웹사이트 적용 방안 (Web Implementation Plan)

웹사이트의 전체적인 UI/UX는 **Deep Navy Blue**를 주 색상으로 채택하며, 모든 데이터 차트 및 상태 표시에는 **System Green**을 일관되게 사용합니다.

1.  **Navigation & Header:**
    *   로고와 메인 메뉴는 Deep Navy Blue 배경 위에 화이트 텍스트로 배치하여 시각적 안정감을 확보합니다.
2.  **Dashboard/Data Visualization:**
    *   모든 주요 지표($PTI$, $ROI$)의 그래프 및 카드에는 **System Green**을 사용하여 '안정성'과 '긍정적인 상태'를 즉시 인지하게 합니다. (예: $PTI$가 높으면 녹색 바, 낮으면 노란색/빨간색 경고 표시).
3.  **Risk Management Module:**
    *   시스템 안전 모드(`system_safe_mode`) 관련 알림이나 경고 메시지는 **System Green**을 사용하여 사용자에게 즉각적인 통제감과 안심을 제공합니다. (Red Alert 색상은 비상 상황에만 사용)
