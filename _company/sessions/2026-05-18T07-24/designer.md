## 💎 디자인 시스템 최종 가이드라인 확정

### 1. 핵심 컬러 팔레트 (Color Palette)
시스템 안정성과 신뢰감을 극대화하기 위해 기존의 색상 코드를 최종 확정하고, 각 상태(안정/경고)에 따른 구분을 명확히 합니다.

| 용도 | 색상 이름 | HEX 코드 | 역할 및 의미 |
| :--- | :--- | :--- | :--- |
| **Primary (신뢰)** | Deep Navy Blue | `#0A1931` | 플랫폼의 깊이, 전문성, 안정성을 상징. 배경 및 주요 텍스트에 사용. |
| **Accent (안정/성장)** | System Green | `#00C853` | 시스템 안전 모드(Safe Mode) 활성화, 성공적인 리스크 관리를 나타내는 핵심 색상. |
| **Neutral (배경)** | Light Gray | `#F4F6F8` | 콘텐츠 영역의 배경으로 사용되어 Navy Blue와 Green을 돋보이게 함. |
| **Warning (경고)** | Amber/Orange | `#FFB300` | 리스크 경고 또는 주의가 필요한 지표에 사용. |
| **Danger (위험)** | Red | `#D50000` | 시스템 안전 모드(System Safe Mode) 작동, 즉각적인 조치가 필요함을 알림. |

### 2. 타이포그래피 스타일 (Typography Style)
데이터 기반의 신뢰감을 유지하며 가독성을 확보합니다.

*   **Font Family:** 산세리프 계열 (예: Pretendard 또는 Inter). 데이터와 시스템 UI에 적합한 명료하고 현대적인 느낌을 강조합니다.
*   **Hierarchy:** 숫자는 볼드 처리하여 지표(PTI, ROI)를 강조하며, 설명 텍스트는 충분한 여백(Whitespace)을 확보하여 복잡한 데이터를 쉽게 이해하도록 설계합니다.

### 3. UI/UX 시각화 전략 (Visualization Strategy)

#### A. 티어 차별화 (Premium vs. Guardian)
두 티어의 차이를 **'안정성의 깊이'**로 명확히 구분하며 시각화합니다.

*   **Premium:** 시스템 안정성(PTI) 지표를 주로 표시하며, 일반적인 리스크 관리 기능을 제공함을 강조합니다. (System Green 사용)
*   **Guardian:** PTI 지표 외에 **실시간 동적 안전장치(Dynamic Safety Locks)**의 상태와 자동 방어 로직의 완벽성을 시각적으로 강조하여 프리미엄 가치를 부여합니다. (Deep Navy Blue 배경 + System Green의 강력한 대비)

#### B. 시스템 안전 모드 시각화 (`system_safe_mode`)
이 기능은 플랫폼의 핵심 차별점입니다.

*   **활성화 상태:** 화면 전체 또는 핵심 지표 주변에 **System Green** 색상의 앰버(Amber) 경고 바를 오버레이하여 즉각적인 주의를 환기시킵니다.
*   **정지 상태 (Safe Mode):** 시스템이 정지되었을 때, 배경색을 **Deep Navy Blue**로 전환하고 중앙에 큰 System Green 텍스트로 "SYSTEM SAFE" 또는 "RISK CONTROL ACTIVE"를 명확하게 표시하여 플랫폼의 통제력을 시각화합니다.

### 4. 고해상도 이미지/배너 파일 최종 사양 및 생성 (Asset Generation Plan)

이 가이드라인을 바탕으로, 이전에 제작된 파일들을 **'시스템 안정성 중심'**으로 재정비하고 통합합니다.

#### A. 유튜브 썸네일 컨셉 (Thumbnail Concepts - 3안 비교 정리)
최종 선택은 **"Guardian 티어의 시스템 안정성 강조"**를 중심으로 합니다.

| 컨셉 | 핵심 시각 요소 | 컬러 조합 | 목적 | 최종 추천 여부 |
| :--- | :--- | :--- | :--- | :--- |
| **1안 (PTI Focus)** | PTI 지표 그래프 + 방패 아이콘 | Navy Blue / System Green | 시스템 안정성 수치 강조. 데이터 기반 신뢰도 어필. | ✅ 높음 |
| **2안 (Guardian Focus)** | 잠금장치/방어막 아이콘 + 붉은 경고 (Red) 요소 최소화 | Deep Navy Blue / System Green | 최고 수준의 리스크 제어(Guardian)를 통한 안전성 강조. | ⭐ 매우 높음 |
| **3안 (ROI vs PTI)** | 상승하는 그래프와 정지된 시스템 UI | Gradient (Green to Blue) | 수익률이 아닌 안정성이 우선임을 은유적으로 표현. | 보통 |

**➡️ 최종 선택:** **2안 (Guardian Focus)**을 메인으로 하되, 1안의 데이터 명료성을 결합하여 제작합니다.

#### B. 배너 파일 가이드라인
*   **Banner Template:** Deep Navy Blue 배경에 System Green의 미니멀한 라인 아트(Line Art)를 사용하여 시스템 안정성의 느낌을 전달합니다.
*   **CTA 강조:** 모든 CTA 버튼은 **System Green**으로 채우고, 텍스트는 흰색 또는 밝은 회색을 사용하여 대비를 극대화합니다.
