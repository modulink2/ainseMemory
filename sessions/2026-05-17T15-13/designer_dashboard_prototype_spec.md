# 대시보드 프로토타입 상세 명세 (Deep Navy Blue & System Green)\n\n## 1. 디자인 원칙\n*   **톤앤매너:** 데이터 기반 신뢰, 리스크 제어형 스마트 투자 플랫폼.\n*   **컬러 팔레트:** Primary: Deep Navy Blue (\#001F3F), Accent: System Green (\#00C853), Warning: Red (\#D50000).\n*   **KPI 우선순위:** PTI(시스템 안정성 지수)가 모든 시각 요소 중 가장 높은 시각적 중요도를 가짐.\n\n## 2. 레이아웃 구조 (Wireframe Outline)\n(사용자에게 제공될 Figma/Sketch의 기본 프레임 구성)\n1. **Header Bar (Navy Blue):** 로고, 사용자 메뉴.\n2. **PTI Control Panel (Focus Area):** PTI 수치와 안전 모드 스위치 배치.\n3. **Core KPI Row:** ROI, Net Value 등 주요 금융 지표 표시.\n4. **Stability Visualization (Main Body):** PTI 추이 라인 차트 및 위험 영역 시각화.\n5. **Portfolio List:** 보유 종목 목록과 개별 리스크 상세 정보.\n\n## 3. 핵심 컴포넌트 상세 디자인\n### A. PTI 강조 카드 (Top KPI)\n*   **배경:** Deep Navy Blue (약간 어두운 그라데이션 적용).\n*   **PTI 값:** 매우 큰 폰트, System Green 색상으로 표시. (예: **85.2%**)\n*   **레이블:** 시스템 안정성 지수 (PTI)\n\n### B. 안전 모드 스위치 (Control Component)\n*   **상태(OFF):** 배경은 Navy Blue, 스위치 색상은 회색.\n*   **상태(ON):** 배경이 System Green으로 바뀌고, 스위치 아이콘이 강조됨. 텍스트는 흰색.\n\n### C. PTI 추이 차트 (Stability Trend Chart)\n*   **축:** X축: 시간, Y축: PTI 값.\n*   **라인 색상 로직:** 라인은 기본적으로 System Green을 사용하며, 위험 임계치(예: 70%)를 초과하면 자동으로 Red로 전환됨. \n*   **시각화 요소:** 경고 영역(Red/Yellow)이 그래프 위에 명확한 배경색으로 표시되어 시각적 경고를 즉시 전달.\n\n## 4. 티어별 차등 디자인 (Basic vs Pro)\n*   **Basic View:** ROI, PTI 요약 통계, 핵심 종목 목록만 노출. (정보 밀도 낮음)\n*   **Pro View:** Basic 정보 외에 시스템 안정성 모듈 상세 분석, 예측 알고리즘 결과(PTI 연관성), 고급 리스크 시뮬레이션 그래프를 추가하여 표시.\n\n이 명세를 기반으로 Figma/Sketch에서 프로토타입 개발을 시작하십시오.
